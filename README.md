今天学习了git部分指令
自己将本地代码推送到一个新创建，空的远程仓库中：
```
git remote add origin xxx(远程地址)
git add .
git commit -m "文字描述"
git push -u origin xxx(分支名)
```
自己拉取远程代码时，进行以下操作：
```
git clone xxx(远程仓库地址)
git add .(暂存自己的代码)
git commit (提交自己代码到本地仓库中)
git pull (这一步可能会需要，当有其他人提交代码后)
git push (推送到远程仓库)
```
